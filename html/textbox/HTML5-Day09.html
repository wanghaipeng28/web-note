4.碰撞检测
   碰撞的四个条件：
		obj1.x + obj1.width >= obj2.x
		obj2.x + obj2.width >= obj1.x
		obj1.y + obj1.height >= obj2.y
		obj2.y + obj2.height >= obj1.y
